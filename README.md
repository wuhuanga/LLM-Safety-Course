### ✒️ 实验提交规范

为确保作业能够被顺利批阅，请大家遵循以下流程：

1. **定位任务**：进入 `2026-Assignments` 文件夹，找到你当前需要完成的实验项目对应目录。
2. **明确要求**：仔细阅读该目录下的说明文档，确保完全理解实验的最终目标与限制条件。
3. **规范提交**：按照文档中的具体要求，将你的实验成果放置在对应目录下，并确保提交内容中不包含任何敏感的个人凭证（如 API Key）。
