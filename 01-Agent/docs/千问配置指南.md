# 千问模型配置指南

本项目使用阿里云千问模型作为LLM。

## 步骤1：申请API密钥

1. 访问 [阿里云百炼控制台](https://dashscope.console.aliyun.com/)
2. 登录或注册阿里云账号
3. 在左侧菜单选择 **API-KEY**
4. 点击 **创建新的API-KEY**
5. 复制生成的API Key

## 步骤2：配置环境变量

在项目根目录创建 `.env` 文件：

```bash
cd D:\NUAA\LLM-Safety-Course\01-Agent
cp .env.example .env
```

然后编辑 `.env` 文件，填入你的API密钥：

```bash
DASHSCOPE_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx
```

## 步骤3：安装依赖

```bash
pip install dashscope
```

## 步骤4：测试API

```bash
python scripts/test_qwen.py
```

如果看到 ✅ 表示配置成功！

## 千问模型说明

| 模型 | 说明 | 适合场景 | 价格 |
|------|------|---------|------|
| **qwen-turbo** | 快速响应，免费额度高 | 快速测试、开发 | 免费 |
| **qwen-plus** | 平衡性能和成本 | 正式开发 | 低成本 |
| **qwen-max** | 最强性能 | 复杂推理任务 | 较高成本 |

### 推荐：
- **开发阶段**：使用 `qwen-turbo`（免费，速度快）
- **正式运行**：使用 `qwen-plus`（性价比高）

## 配置模型

在 `config.yaml` 中修改：

```yaml
llm:
  provider: dashscope
  model: qwen-turbo  # 改成你想要的模型
```

## 免费额度

- 新用户有免费额度
- qwen-turbo 免费额度最高
- 查看额度：[百炼控制台](https://dashscope.console.aliyun.com/billing)

## 常见问题

### Q1: 提示"API密钥无效"
A: 检查 `.env` 文件中的密钥是否正确，注意不要有多余空格

### Q2: 提示"余额不足"
A: 充值账户或使用免费模型 qwen-turbo

### Q3: 如何查看用量？
A: 访问 https://dashscope.console.aliyun.com/billing
