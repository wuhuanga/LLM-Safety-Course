## 1. 系统设计

### 1.1 系统架构

本系统基于ReAct（Reasoning + Acting）框架，实现了记忆增强型文本交互智能体。

```
┌─────────────────────────────────────────────────────────┐
│                      用户交互层                            │
└──────────────────────┬──────────────────────────────────┘
                       │
┌──────────────────────▼──────────────────────────────────┐
│                    ReAct Agent                            │
│  ┌──────────────────────────────────────────────────┐  │
│  │              思考(Thought) → 行动(Action)         │  │
│  └────────────────────┬─────────────────────────────┘  │
│  ┌────────────────────▼─────────────────────────────┐  │
│  │              记忆检索(RAG)                        │  │
│  └────────────────────┬─────────────────────────────┘  │
│  ┌────────────────────▼─────────────────────────────┐  │
│  │              动作验证与转换                        │  │
│  └────────────────────┬─────────────────────────────┘  │
└───────────────────────┼───────────────────────────────────┘
                        │
┌───────────────────────▼───────────────────────────────────┐
│                    记忆管理层                             │
│  ┌──────────────────────────────────────────────────┐  │
│  │         语义记忆 | 情景记忆 | RAG检索              │  │
│  └──────────────────────────────────────────────────┘  │
└───────────────────────┼───────────────────────────────────┘
                        │
┌───────────────────────▼───────────────────────────────────┐
│                 ScienceWorld环境                          │
└───────────────────────────────────────────────────────────┘
                        │
┌───────────────────────▼───────────────────────────────────┐
│                  千问模型 (qwen-max)                       │
└───────────────────────────────────────────────────────────┘
```

### 1.2 核心模块

| 模块 | 功能 | 技术方案 |
|------|------|---------|
| **环境接口** | ScienceWorld封装 | `getValidActionObjectCombinations()` |
| **LLM模块** | 推理和嵌入 | DashScope API |
| **语义记忆** | 存储环境知识 | 向量嵌入 + 余弦相似度 |
| **情景记忆** | 存储历史轨迹 | 时间序列 + 成功标记 |
| **RAG检索** | 相关记忆检索 | 上下文过滤 + 相似度排序 |
| **ReAct Agent** | 推理-行动循环 | Thought-Action循环 |
| **动作验证** | 动作转换 | 智能匹配 + 失败避免 |

## 4. 关键技术

### 4.1 ReAct推理循环

系统采用ReAct框架，通过思考-行动循环实现任务执行：

1. **思考**: 分析当前状态，检索相关记忆
2. **行动**: 选择并执行动作
3. **观察**: 获取环境反馈
4. **更新**: 记录经验到记忆

### 4.2 记忆检索优化

为提高检索效率，实现了：
- 上下文过滤（只检索位置相关的记忆）
- 重要性排序（失败的记忆优先级更高）
- 动作历史管理（避免重复失败）

### 4.3 动作智能选择

LLM面对大量动作时难以选择，通过：
- 动作分类（移动、探索、任务相关）
- 优先级排序（根据当前位置动态调整）
- 失败动作过滤
